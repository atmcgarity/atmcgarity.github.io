My Projects

PacMan Factory

This project allows you to spawn numerious PacMen on the browser window and amke them to move. They will fly around and hit the borders of the brower causing them bounce around. This was made using JavaScript and HTML.

Eye Tracker

The Eye Tracker project shows two eyes when loaded onto the brower window, the pupils will follow the movement of your cursor. This porject was the first of the three projects I've presented in which CSS was involved.

Map Animation

The Map Animation project shows a marker stopping at the various position. These stops represent the various bus stops between the MIT and Harvard Campuses in Boston. This project, like the eye tracker, uses JavaScript, CSS, and HTML but it is the first project in which API's are present.
